# Command Line

## Commands to Remember

**macOS & GNU/Linux**

* list files and folders: `ls` and `ls -a`.
* create files: `touch`.
* create folders: `mkdir`.
* navigate: `cd`.
* remove files: `rm`.
* remove folders: `rm -rf`.
* read file contents: `cat`.


**would add**
* write and *read* file contents: `nano`.
